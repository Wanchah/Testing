Variables and Expressions

A variable is a symbol (usually a letter) that represents a number that can change. For example, in the expression 2x + 3, x is a variable.

Example:

If x = 5, then 2x + 3 = 2(5) + 3 = 10 + 3 = 13

Solving Equations

To solve an equation, we need to find the value of the variable that makes the equation true.

Example:

Solve: 3x + 7 = 22

Step 1: Subtract 7 from both sides

3x = 15

Step 2: Divide both sides by 3

x = 5
